NoteUP

NoteUP is a simple web application where users can add notes to document their daily life. It’s built with HTML, CSS, Python, and Flask. This was my first project from the Udemy course Web Developer Bootcamp with Flask and Python in 2024 and at the same time my first professional step towards my full-stack developer journey.

Why was it written?

This app was mainly written for the sake of taking the course. I consider even smallest step towards advancement a victory. Even though this app is small and relatively easy to make I was sure there is something I could learn from this experience.

What I learned?

Even before developing this app, I knew some of the technologies used. I still learned new things — for example, using Figma to create clear, responsive designs and prototypes before writing any code. I’m also excited to use these skills in future projects and collaborate effectively with teammates.

Technologies used

  •  Python/Flask
  •  HTML
  •  CSS
  •  MongoDB

Future

While I plan to focus on new projects, if I were to expand this app further, I would add a user system to allow personalized accounts and note management. Each project builds on the last, so future developments will continue to improve my skills and approach.
